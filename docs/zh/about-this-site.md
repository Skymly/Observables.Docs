# 关于本站

**Observables.Docs** 是 [Observables](https://github.com/Skymly/Observables) 的用户文档站点。

| 内容 | 地址 |
|------|------|
| 源码与 CI | [github.com/Skymly/Observables](https://github.com/Skymly/Observables) |
| 示例 | [github.com/Skymly/Observables.Samples](https://github.com/Skymly/Observables.Samples) |
| 本站 | [github.com/Skymly/Observables.Docs](https://github.com/Skymly/Observables.Docs) |

正文为 **English**（`docs/`）与 **简体中文**（`docs/zh/`）。贡献者细节见主仓库 `AGENTS.md` 与各子项目 README，后续会逐步迁移到本站。

使用 [VitePress](https://vitepress.dev/) 构建，通过 GitHub Pages 发布（`base` 为 `/Observables.Docs/`）。
