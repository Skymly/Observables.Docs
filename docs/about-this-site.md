# About this site

**Observables.Docs** is the user-facing manual for the [Observables](https://github.com/Skymly/Observables) project.

| Item | Location |
|------|----------|
| Source code & CI | [github.com/Skymly/Observables](https://github.com/Skymly/Observables) |
| Samples | [github.com/Skymly/Observables.Samples](https://github.com/Skymly/Observables.Samples) |
| This site | [github.com/Skymly/Observables.Docs](https://github.com/Skymly/Observables.Docs) |

Content is written in **English** (`docs/`) and **简体中文** (`docs/zh/`). Generator-specific contributor notes remain in the main repository `AGENTS.md` and per-project README files until migrated here.

Built with [VitePress](https://vitepress.dev/). Deployed to GitHub Pages at `/Observables.Docs/` base path.
